About Me

I am a highly motivated Software Engineer with a strong foundation in full-stack development, focused on building scalable, user-friendly web and mobile applications. From front-end interfaces to back-end systems, I enjoy solving real-world problems through clean, scalable code and intuitive design. Over time, I've built and contributed to various projects across frontend, backend, and mobile platforms.

Core Strengths

Frontend Development: Proficient in crafting responsive and accessible user interfaces using HTML, CSS, Tailwind, Bootstrap and JavaScript.

Backend Development: Solid understanding of server-side development with Node.js, Express, and REST APIs.

Mobile Development: Building Android applications using Java and Android Studio, with a focus on API integration and user experience.

Version Control & Deployment: Comfortable using Git, GitHub, and deploying applications via Netlify and GitHub Pages.

Problem Solving: Passionate about writing clean code, debugging, and building efficient solutions with attention to detail.
